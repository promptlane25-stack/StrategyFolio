Step-by-Step: Getting It Live Tonight

Step 1
Set Up Calendly (Free — 20 minutes)
  1. Go to calendly.com and create a free account using the garage's email (info@martleshamservicecentre.co.uk)
  2. Click "New Event Type" → choose "One-on-One"
  3. Create 4 event types (one per service):
    • MOT Test — 60 minutes
    • Interim Service — 90 minutes
    • Full Service — 120 minutes
    • Repairs / Other — 60 minutes
  4. Set availability: Monday–Friday, 8:00am–5:30pm
  5. In each event type, add a custom question: "Vehicle registration number" (required)
  6. Your Calendly username will be something like: calendly.com/martlesham-msc
Step 2
Add Your Calendly Username to the Website File
  1. Open the file martlesham_service_centre.html in any text editor (Notepad on Windows, TextEdit on Mac)
  2. Press Ctrl+F (or Cmd+F on Mac) and search for: YOUR_CALENDLY_USERNAME
  3. Replace it with your actual Calendly username. For example:

    Change: https://calendly.com/YOUR_CALENDLY_USERNAME
    To: https://calendly.com/martlesham-msc
  4. Save the file
⚠️ Until you do this step, the website will show a simple booking request form instead of the Calendly calendar. It still works — it just isn't as slick.
Step 3 — Option A (Recommended)
Upload Via FTP / cPanel (Replace the Old Site)

The current site is hosted by Netphobia. You need hosting access from them.

  1. Email Netphobia (their email is on the current site footer) and ask: "Can you please provide FTP login details or cPanel access so we can update the website?"
  2. Once you have FTP details, download FileZilla (free — filezilla-project.org)
  3. Connect using the FTP details provided
  4. Navigate to the public_html folder (or www folder)
  5. You'll see the existing index.html file. Rename it to index_old.html (keeps it as a backup)
  6. Upload the new martlesham_service_centre.html file and rename it to index.html
  7. The new site is live
Step 3 — Option B (Tonight, No Website Access Needed)
Get Calendly Live Before the Site Is Even Touched

You don't need to wait for website access to start using the Calendly booking system. Do this tonight:

  1. Complete Step 1 above (set up Calendly)
  2. Share the Calendly link directly with the garage owner: calendly.com/martlesham-msc
  3. They can immediately add it to:
    • Their Google Business Profile (add as a "Book" button)
    • Their email signature
    • WhatsApp messages to customers
    • Their Facebook page if they have one
  4. The website upload can happen next week once you have FTP access
✓ This means they can start taking online bookings tonight — before the new website is even live.

What to Tell Them About Maintenance

The website is a single HTML file. It is very easy to maintain. Specifically:

  • To change a price: Open the file in Notepad, find the old price with Ctrl+F, type the new one, save, re-upload via FTP. Takes 5 minutes.
  • To add a new service: Copy an existing service card block in the HTML and change the text. 10 minutes.
  • To add a blog: For proper SEO blogging, the recommendation is to migrate to WordPress (a one-time 2–3 hour job you can charge £150–200 for). This gives a proper content management system so you don't need to touch HTML every time.
  • To update Calendly availability (e.g. holiday closures): Log into Calendly → go to availability settings → mark days as unavailable. No website change needed — Calendly handles it automatically.
